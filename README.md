Training Planner 
Автор: Волошенко Глеб
Дата: 2026-04-29
Описание 
Приложение для планирования тренировок с возможностью: * добавления тренировок с
указанием даты, типа и длительности; * фильтрации записей по типу тренировки и дате; *
сохранения данных в формате JSON; * загрузки данных при запуске приложения. 
Примеры использования 
Добавление тренировки 
Введите дату в формате YYYY-MM-DD (например, 2024-01-15).
Выберите тип тренировки из выпадающего списка.
Укажите длительность в минутах (положительное число).
Нажмите кнопку «Добавить тренировку». 
Фильтрация данных 
Выберите тип тренировки в фильтре «Тип».
Введите часть даты в фильтре «Дата» (например, 2024-01 для всех тренировок в январе 2024
года).
Нажмите кнопку «Применить фильтры» для обновления таблицы. 
Тестирование приложения 
Тест 1: Добавление корректной тренировки 
Ввод: 
Дата: 2024-01-15 
Тип: «Бег» 
Длительность: 60 
Ожидаемый результат: тренировка успешно добавлена в таблицу, данные сохранены в
data.json. 
Тест 2: Некорректный формат даты 
Ввод: 
Дата: 15-01-2024 
Тип: «Плавание» 
Длительность: 45 
Ожидаемый результат: появляется сообщение об ошибке «Неверный формат даты.
Используйте YYYY-MM-DD». 
Тест 3: Отрицательная длительность 
Ввод: 
Дата: 2024-01-16Тип: «Велоспорт» 
Длительность: -30 


Ожидаемый результат: появляется сообщение об ошибке «Длительность должна быть
положительным числом». 
Тест 4: Фильтрация по типу 
Действие: выберите «Бег» в фильтре типа и нажмите «Применить фильтры».
Ожидаемый результат: в таблице отображаются только тренировки типа «Бег». 
Тест 5: Фильтрация по дате 
Действие: введите 2024-01 в фильтре даты и нажмите «Применить фильтры».
Ожидаемый результат: в таблице отображаются тренировки за январь 2024 года.
